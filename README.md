# Asteroids-Vue
An app that displays asteroids and allows users to like and unlike their favorite asteroids.  

### Features
+ Responsive Design
+ Beautiful UI
+ Like/Unlike feature
+ Full Auth Implementation
+ Search and Find Asteroids
+ Live Updates
+ Error Handling

##### Built With
+ HTML5
+ SASS
+ Vue
+ Firebase
+ NASA API (Search)

### Getting Started
clone repo

`cd` into folder and run `yarn install`

run `yarn serve`
